# Arrow Hooks

> 一个实用的 React Hooks 集合，专注于提供高质量、类型安全的 React Hooks

## 特性

- 🎯 **专注于实用性** - 都是实际开发中高频常见的功能需求
- 📦 **基于 Vite 构建** - 快速的开发体验和优化的构建输出
- ✅ **Vitest 单元测试** - 所有 Hook 都经过完整的单元测试验证
- 🔷 **TypeScript 支持** - 完整的类型定义和类型推断
- 🚀 **ESM 支持** - 现代化的模块系统，支持 Tree Shaking
- ⚙️ **客户端优先** - 目前专注于客户端渲染（SSR 支持正在计划中）

## Hook 列表

### DOM 操作
- [useEventListener](/hooks/use-event-listener) - 类型安全的 DOM 事件监听器
- [useHover](/hooks/use-hover) - 检测元素悬停状态
- [useResizeObserver](/hooks/use-resize-observer) - 监听 DOM 元素尺寸变化
- [useIntersectionObserver](/hooks/use-intersection-observer) - 监听元素可见性变化

### BOM 操作
- [useWindowSize](/hooks/use-window-size) - 监听窗口尺寸变化
- [useMediaQuery](/hooks/use-media-query) - 响应媒体查询变化

### 存储与数据
- [useCookie](/hooks/use-cookie) - 操作浏览器 Cookie

### 多媒体与交互
- [useKeyboardEvent](/hooks/use-keyboard-event) - 处理键盘事件
- [useInfiniteScroll](/hooks/use-infinite-scroll) - 实现无限滚动

## 快速开始

安装依赖：

```bash
pnpm add arrow-hooks
```

基础使用示例：

```tsx tab="React 函数组件"
import { useCookie, useWindowSize } from 'arrow-hooks'

function App() {
  const [theme, setTheme] = useCookie('theme', 'light')
  const { width, height } = useWindowSize()

  return (
    <div>
      <p>当前主题: {theme}</p>
      <p>窗口尺寸: {width} × {height}</p>
      <button onClick={() => setTheme(theme === 'light' ? 'dark' : 'light')}>
        切换主题
      </button>
    </div>
  )
}
```

## 开发理念

Arrow Hooks 专注于：
- **实用性优先** - 收录在实际项目高频率场景的 Hook
- **质量保证** - 每个 Hook 都有较完整的测试覆盖
- **类型安全** - 提供完整的 TypeScript 支持
- **开箱即用** - 简单的 API 设计，即插即用
