# useHover

用于检测 DOM 元素是否处于 hover 状态的 Hook。

## 基础用法

```tsx
import React, { useRef } from 'react';
import { useHover } from 'arrow-hooks';

export default () => {
  const elementRef = useRef(null);
  const isHovered = useHover(elementRef);

  return (
    <div
      ref={elementRef}
      style={{
        padding: '20px',
        backgroundColor: isHovered ? '#e6f7ff' : '#ffffff',
        border: '1px solid #91d5ff',
        transition: 'all 0.3s',
      }}
    >
      {isHovered ? '鼠标悬停中！' : '将鼠标悬停在此处'}
    </div>
  );
};
```

## API

```typescript
const isHovered = useHover(elementRef);
```

### 参数

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| elementRef | 需要监听 hover 状态的 DOM 元素的 ref | `RefObject<HTMLElement>` | - |

### 返回值

| 参数 | 说明 | 类型 |
| --- | --- | --- |
| isHovered | 元素是否处于 hover 状态 | `boolean` |
